# Blank Screen

[简体中文](README.zh-CN.md)

A minimalist Android application that displays a completely black screen and prevents the device from sleeping.

## Features

- **Full Black Screen**: Pure black background covering the entire screen, including status and navigation bars.
- **Always On**: Uses `FLAG_KEEP_SCREEN_ON` to ensure the screen never dims or turns off while the app is active.
- **Quick Settings Tile**: Includes a "Blank Screen" tile in the notification panel for quick access.
- **Adaptive Icon**: A minimalist black-on-black adaptive icon.
